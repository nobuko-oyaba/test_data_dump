# test_data_dump
Microsoft Sql Serverのテーブルデータのダンプ用スクリプト。  
コンソールとスクリプト直下のfile.txtにSQL文実施し、その結果をダンプします。

|   id | xxxxxxxxxxxx_xx   | xxxxxx   | xxxx     |   xxx | xxx      | xxx      |   xxxxxxxxxxxxxxxxxxxxxxxx |

| 1101 | XXXXXXXXXXXX      | 00000037 | XXXX0037 |     0 | テスト68 | テスト68 |                          0 |
+------+-------------------+----------+----------+-------+----------+----------+----------------------------+
| 1102 | XXXXXXXXXXXX      | 00000038 | XXXX0038 |     0 | テスト69 | テスト69 |                          1 |
+------+-------------------+----------+----------+-------+----------+----------+----------------------------+
| 1103 | XXXXXXXXXXXX      | 00000039 | XXXX0039 |     0 | テスト70 | テスト70 |                          2 |
